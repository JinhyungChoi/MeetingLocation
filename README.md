# 만나는 장소 정해주는 앱
## 서비스 소개
- 사용자들이 만나는 장소를 정할 때 쉽고 빠르게 정할 수 있도록 도와주는 서비스입니다. 사용자가 입력한 장소들을 기반으로 거리적 중점이 아닌 대중교통 상황을 반영한 시간적 중점을 만나는 장소로 추천해줍니다.
